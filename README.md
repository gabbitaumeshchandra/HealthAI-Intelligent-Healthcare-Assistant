# 🏥 HealthAI – Intelligent Healthcare Assistant Using IBM Granite

HealthAI is an AI-powered healthcare assistant designed to improve patient management, disease prediction, medical guidance, and healthcare analytics through an intelligent digital platform.

This project combines healthcare workflows with AI-powered assistance using Python, Streamlit, SQLite, and IBM Granite-ready architecture to create a professional healthcare SaaS-style platform.

---

## 🔐 Login System

# ⚙ Installation

## Step 1: Clone Repository

```bash
git clone https://github.com/yourusername/HealthAI-Intelligent-Healthcare-Assistant.git
cd HealthAI-Intelligent-Healthcare-Assistant
```

## Step 2: Install Dependencies

```bash
pip install -r requirements.txt
```

## Step 3: Run Application

```bash
streamlit run app.py
```
